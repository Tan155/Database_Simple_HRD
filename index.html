รายละเอียดโครงสร้างตาราง (Data Dictionary)

ตาราง Employees (ข้อมูลพนักงาน)

ชื่อฟิลด์ (Field)คำอธิบาย (Description) / การใช้งาน
รหัสพนักงาน (Employee ID) [PK]ใช้เป็น Username ในการล็อกอินเข้าสู่ระบบ
รหัสผ่าน (Password)พนักงานสามารถตั้งเองและต้องเปลี่ยนทุก 3 เดือน
ชื่อ-นามสกุล / อีเมลสำหรับพนักงานระดับ Officer ขึ้นไปที่มีอีเมลบริษัท
ระดับพนักงาน (Level/Role)เช่น พนักงานหน้างาน, Officer เพื่อใช้คัดกรองการลงทะเบียนคอร์ส
รหัสสาขา/โรงงาน (Location ID) [FK]ระบุว่าพนักงานอยู่โรงงานใด หรือออฟฟิศไหน
รหัสแผนก (Department ID) [FK]เช่น HR, Finance เพื่อดึงชุดคำถามประเมินให้ตรงแผนก

ตาราง Locations (ข้อมูลสาขา)

ชื่อฟิลด์คำอธิบาย
Location ID [PK]รหัสอ้างอิงสถานที่
Location Nameเก็บข้อมูลสาขาทั้ง 6 โรงงาน และ 2 ออฟฟิศ

ตาราง Departments (ข้อมูลแผนก)

ชื่อฟิลด์คำอธิบาย
Department ID [PK]รหัสอ้างอิงแผนก
Dept_Name / Positionเก็บข้อมูลแผนกและตำแหน่งต่างๆ ในองค์กร

ตาราง Competencies (ข้อมูลทักษะ)

ชื่อฟิลด์คำอธิบาย
Competency ID [PK]รหัสอ้างอิงทักษะ
Competency Nameชื่อทักษะ (เช่น Communication, Digital Marketing, Data Analytic)

ตาราง Position_Competencies

ชื่อฟิลด์คำอธิบาย
Position ID [FK]รหัสตำแหน่งงาน
Competency ID [FK]รหัสทักษะที่เกี่ยวข้อง
Required Scoreระดับคะแนนที่ต้องการ (เช่น ต้องการ Communication ระดับ 4)

ตาราง Assessment_Templates (ชุดคำถามประเมิน)

ชื่อฟิลด์คำอธิบาย
Department ID [FK]แยกชุดคำถามให้แตกต่างกันตามแผนก
Questionsข้อคำถามที่ใช้ประเมิน

ตาราง Assessment_Scores (ผลการประเมิน)

ชื่อฟิลด์คำอธิบาย
Employee ID [FK]รหัสพนักงานผู้รับการประเมิน
Competency ID [FK]รหัสทักษะที่ถูกประเมิน
Self Assessment Scoreคะแนนประเมินตนเอง
360° Scoreคะแนนจากหัวหน้างานและเพื่อนร่วมงาน
Skill Gapคำนวณจาก Required Score ลบด้วยคะแนนประเมินที่ได้

ตาราง IDP_Goals (เป้าหมายการพัฒนา)

ชื่อฟิลด์คำอธิบาย
Employee ID [FK]รหัสพนักงานเจ้าของเป้าหมาย
Short-term Goalเป้าหมายระยะสั้น เช่น ภายใน 1 ปี
Long-term Goalเป้าหมายระยะยาว เช่น 3-5 ปี
Skill Detailsรายละเอียดทักษะที่จะพัฒนา
Approval Statusรอดำเนินการ, หัวหน้างานอนุมัติแล้ว หรือต้องแก้ไข

ตาราง Courses (ข้อมูลหลักสูตร)

ชื่อฟิลด์คำอธิบาย
Course Nameชื่อคอร์ส
Course TypeOnline (E-learning) หรือ On-site
Category/Tagเพื่อใช้เชื่อมโยงกับ Skill Gap และแนะนำคอร์สเรียน
Target Levelระดับพนักงานที่อนุญาตให้เรียน
Assessment Linksลิงก์ Pre-test, Post-test และ Survey ลิงก์แบบประเมิน
Clipไฟล์หรือลิงก์วิดีโอคลิปสำหรับการเรียน

ตาราง Course_Sessions

ชื่อฟิลด์คำอธิบาย
Course ID [FK]รหัสคอร์ส
Session Date/Timeวันที่เรียน (เช่น กรุ๊ป 1: 1/7, กรุ๊ป 2: 2/7)
Registration TypeFirst Come, First Serve หรือระบบโควตา (HR ยัดรายชื่อ)
Total Capacityจำนวนที่นั่งรวมทั้งหมดในรอบนั้น

ตาราง Session_Quotas

ชื่อฟิลด์คำอธิบาย
Session ID [FK]รหัสรอบการอบรม
Location ID [FK]รหัสสาขา/โรงงาน
Allocated Seatsจำนวนที่นั่งที่ได้รับจัดสรร เช่น 4 คนต่อ 1 โลเคชั่น

ตาราง Enrollments (ประวัติการลงทะเบียนและการเรียน)

ชื่อฟิลด์คำอธิบาย
Employee ID [FK]รหัสพนักงานที่ลงทะเบียน
Course ID / Session ID [FK]รหัสคอร์ส และรหัสรอบการอบรมที่เลือก
Registration Statusสถานะ: สมัครแล้ว, ยกเลิกล่วงหน้า, แอดมินอนุมัติ (Approved)
Completion Statusเรียนจบ, ทำ Pre-test/Post-test/Survey ครบถ้วน (ใช้แทนเช็คชื่อ)
Certificate Statusระบบออกใบเซอร์ให้เมื่อเรียนจบตามเงื่อนไขแอดมิน
